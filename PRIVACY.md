===========================================

Последнее обновление: 29 мая 2026 г.

Настоящая Политика конфиденциальности описывает, как веб-сайт KirDev Studio взаимодействует с данными пользователей.

1. Сбор персональных данных
   Официальный сайт KirDev Studio является полностью статическим сайтом. Мы:
   - НЕ собираем, НЕ храним и НЕ обрабатываем ваши персональные данные (имена, адреса электронной почты, номера телефонов и т.д.).
   - НЕ требуем регистрации или авторизации для использования сайта или скачивания программ.

2. Файлы Cookie и аналитика
   - Наш сайт НЕ использует файлы cookie (cookies) для отслеживания ваших действий.
   - Мы НЕ внедряем сторонние аналитические системы (такие как Google Analytics, Яндекс.Метрика и др.), которые могут отслеживать ваше поведение в сети.

3. Скачивание программного обеспечения
   - Все программы на сайте загружаются либо напрямую в виде локальных файлов, либо перенаправляют вас на официальные релизы в сервисе GitHub.
   - Любое скачивание файлов через GitHub регулируется политикой конфиденциальности платформы GitHub.

4. Хостинг
   - Сайт хостится на платформе GitHub Pages. При посещении сайта серверы GitHub могут автоматически регистрировать стандартную техническую информацию (например, ваш IP-адрес, тип браузера, операционную систему). Эти данные обрабатываются непосредственно GitHub в целях обеспечения безопасности и стабильности работы сервиса в соответствии с их политикой конфиденциальности.

PRIVACY POLICY
==============

Last updated: May 29, 2026

This Privacy Policy describes how the KirDev Studio website handles user data.

1. Collection of Personal Data
   The official KirDev Studio website is a static site. We:
   - DO NOT collect, store, or process any of your personal data (such as names, emails, phone numbers, etc.).
   - DO NOT require registration or login to browse the site or download software.

2. Cookies and Tracking
   - This website DOES NOT use cookies or tracking technologies.
   - We DO NOT embed third-party analytics (like Google Analytics, Yandex.Metrica, etc.) that track your online behavior.

3. Software Downloads
   - All downloads are served either as local static files or link directly to official releases on GitHub.
   - Any interaction with GitHub Releases is governed by GitHub's own Privacy Policy.

4. Hosting
   - This website is hosted on GitHub Pages. When you visit the site, GitHub servers may automatically log standard web request information (e.g., IP address, browser type, OS). This logging is handled directly by GitHub for security and operational purposes in compliance with GitHub's Privacy Statement.
